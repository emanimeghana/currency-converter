# Currency Converter

## Description
This is a beginner-friendly web application that converts one currency into another using real-time exchange rates.

## Technologies Used
- HTML
- CSS
- JavaScript
- Exchange Rate API

## Features
- Convert between multiple currencies
- Real-time exchange rates
- Simple and clean UI

## How to Run
1. Download or clone the repository
2. Open index.html in any browser
3. Enter amount and select currencies
